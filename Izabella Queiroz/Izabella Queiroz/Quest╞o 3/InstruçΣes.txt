HTML Mobile

A maioria dos acessos web no mundo são via celular. Todos os sites devem estar capacitados para carregar o que é chamado de versão mobile - uma versão do site que é facilmente visível em celulares.

O arquivo "index.html" contém um exemplo de página que funciona bem em computadores, mas ao visualizá-la em um celular, a leitura do texto e da imagem é dificultada, requerendo um "zoom" do usuário.

2 - UTILIZAÇÃO DO BOOTSTRAP
O Bootstrap é uma famosa biblioteca frontend. Ela permite desenvolvimento acelerado de páginas, utilizando classes pré-definidas para padronizar o look-and-feel dos sites.
Na página em questão, o Bootstrap já está carregado.
No HTML, existem dois blocos de configuração (principal e secundario) um abaixo do outro. Cada um com 3 configurações.
Utilizando as classes do Bootstrap, deixe os dois blocos lado-a-lado na tela.

Não se esqueça de consultar a documentação do Bootstrap: https://getbootstrap.com/docs/3.3/css/#grid
